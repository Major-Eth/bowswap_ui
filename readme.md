## Setup  

1. Install the dependencies with `yarn`  
2. Create a `.env` file in the root directory with
```
ALCHEMY_KEY=YOUR_ALCHEMY_KEY
```  
3. Start the dev project with `yarn run dev`
4. [Go check the swap](http://localhost:3000)
